Name and describe 4 different kinds of positioning in CSS.

Answer:

Static-- Default flow. Elements render in the order in which they appear in document.
Absolute-- Not part of the document flow. Position relative to the first positioned ancestor element.
Fixed--Not part of the document flow. The element is positioned relative to the browser window.
Relative--Part of the normal flow. Positioned relative to its normal position.
